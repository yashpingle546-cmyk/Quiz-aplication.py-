🚀 Features
Interactive Interface: Easy and engaging user experience.
Score Tracking: Real-time score updates after every question.
Beginner Friendly: A great project for those learning Python fundamentals.

🧠 Concepts Used
Python Programming
Tkinter: Used for creating the Graphical User Interface (GUI).
Logic: Implemented using conditional statements and loops to manage quiz flow and answer validation.

🛠 How to Run
Ensure you have Python installed on your system.
Clone or download this repository.
Open your terminal or command prompt in the project folder.
Run the application using the command: python main.py (replace main.py with your actual filename).
